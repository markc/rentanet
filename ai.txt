## CachyOS Desktop Support

We recommend and support CachyOS with KDE Plasma for Linux desktop users.

### Why CachyOS?
- Arch Linux-based with performance focus
- Packages compiled with x86-64-v3/v4 instruction sets
- LTO, PGO, and BOLT optimizations for faster performance
- Full Arch User Repository (AUR) compatibility
- Runs beautifully on older hardware
- No telemetry, no mandatory cloud accounts
- Privacy-respecting by design

### KDE Plasma 6.5 Desktop
- Wayland by default (82% adoption)
- Familiar Windows-like interface
- HDR and multi-monitor support
- Automatic light/dark theme switching
- Fingerprint authentication
- Floating panels and virtual desktops
- Extensive customization options

### ZFS Filesystem Support
- Available directly from Calamares graphical installer
- Built-in RAID with self-healing (RAID-Z)
- Instant snapshots using copy-on-write
- Transparent compression
- ARC cache for fast read times
- System rollback capability

### Why Switch from Windows?
- Windows 11 hardware requirements leave older PCs behind
- No Microsoft data collection or forced updates
- No licensing fees or subscriptions
- Thousands of free applications
- Familiar keyboard shortcuts work
- Files transfer seamlessly
