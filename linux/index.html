About CachyOS

Ready to break free from Windows?

CachyOS is a lightweight, Arch Linux-based distribution that emphasizes performance, security, and user-friendliness. Built with optimized kernels and modern tooling, it delivers exceptional speed while maintaining full compatibility with the extensive Arch User Repository (AUR). All packages are compiled with x86-64-v3/v4 instruction sets and LTO optimization, with core packages further enhanced using PGO and BOLT — resulting in measurably faster performance than generic Linux distributions.

Why make the switch?

Windows 11's hardware requirements have left millions of perfectly good computers behind. Microsoft's aggressive data collection, forced updates, and built-in advertising have frustrated users worldwide. Meanwhile, CachyOS runs beautifully on older hardware, respects your privacy completely, and puts you in control of your own computer. No telemetry, no mandatory cloud accounts, no subscriptions — just a fast, stable operating system that works for you.

KDE Plasma: The modern desktop

We recommend the KDE Plasma edition of CachyOS — now at version 6.5, it's the most polished and feature-rich Linux desktop available. Plasma offers a familiar Windows-like experience with a taskbar, start menu, and system tray, but with far greater customization. Wayland is now the default display protocol, bringing smooth animations, HDR support, and excellent multi-monitor handling. Features like automatic light-to-dark theme switching, fingerprint authentication, floating panels, and virtual desktops make it a joy to use daily. Over 82% of KDE users now run Wayland, and the experience is rock-solid.

Enterprise-grade ZFS filesystem

CachyOS is one of the few distributions offering ZFS directly from the Calamares graphical installer — no command-line required. ZFS brings features previously found only in enterprise storage: built-in RAID with self-healing (RAID-Z), instant snapshots using copy-on-write, transparent compression to save disk space, and the legendary ARC cache that delivers blazing-fast read times on frequently accessed files. Roll back your entire system to any previous snapshot in seconds. For users who value their data, ZFS is the gold standard.

It's easier than you think

CachyOS installs in under an hour using the friendly Calamares installer and comes with everything you need: Firefox for browsing, LibreOffice for documents, and thousands of free applications one click away. No subscriptions, no licensing fees, no bloatware. Your familiar keyboard shortcuts work, your files transfer seamlessly, and the learning curve is minimal.

We're here to help

We provide full support for CachyOS with KDE Plasma — from initial installation to ongoing maintenance. Whether you need help migrating from Windows, setting up ZFS snapshots, configuring specific software, or just want guidance as you explore your new system, we've got 28+ years of Linux experience ready to assist you.

Step by Step

How to Install CachyOS

Simple guide to replace Windows with CachyOS

Back up everything important — photos, documents, anything you want to keep. Copy them to an external drive or cloud storage. Installing CachyOS will erase Windows.

Before You Start

You'll need:

  • A USB stick (8GB or larger)
  • About 30-60 minutes

Step 1: Download CachyOS

  1. Go to cachyos.org
  2. Click Download
  3. Choose KDE Plasma (it's the most Windows-like)
  4. Save the .iso file somewhere easy to find

Tip: The file will save to your Downloads folder. It'll be called something like cachyos-kde-linux-250601.iso

Step 2: Create a Bootable USB

  1. Download Rufus from rufus.ie (it's free)
  2. Plug in your USB stick
  3. Open Rufus
  4. Click SELECT and choose the CachyOS .iso you downloaded
  5. Click START and accept any prompts
  6. Wait for it to finish (a few minutes)

Step 3: Boot from the USB

  1. Restart your computer with the USB plugged in
  2. As it starts, repeatedly tap F12, F2, or Del (depends on your computer) to get the boot menu
  3. Select your USB stick from the list
  4. CachyOS will load — this takes a minute or two

Step 4: Install CachyOS

  1. Once loaded, click Install CachyOS
  2. Choose your language and location
  3. Choose your keyboard layout
  4. When asked about disk/partitioning, select Erase disk (the simple option)
  5. Create your username and password
  6. Click through to start installation
  7. Wait 10-20 minutes for it to finish
  8. Click Restart and remove the USB when prompted

Step 5: First Boot

Your computer will restart into CachyOS. Log in with the password you created.

Finding your way around:

  • Bottom-left corner = Start menu equivalent
  • Dolphin = File manager (like File Explorer)
  • Firefox = Web browser
  • Software center = For installing new apps
